# EDA Storyboard Project

## Author
Rupam Mahato

## Project Overview
This project demonstrates an end-to-end Exploratory Data Analysis (EDA) workflow using Python. The objective is to analyze student performance data, identify patterns, and communicate insights through effective visualizations.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Visualizations Included
1. Histogram – Marks Distribution
2. Box Plot – Outlier Analysis
3. Bar Chart – Average Marks by City
4. Scatter Plot – Marks vs Rank
5. Correlation Heatmap
6. Interactive Plotly Visualization

## Key Insights
- Most students scored between 70 and 90 marks.
- No significant outliers were observed in the dataset.
- Student performance varies across different cities.
- Higher marks generally result in better ranks.
- Correlation analysis revealed strong relationships between academic performance metrics.
